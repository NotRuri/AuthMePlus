# AuthMePlus

**AuthMePlus** is a Paper plugin designed to streamline authentication for premium Minecraft players.

It is a fork of [PremiumAuthBypass](https://github.com/akanoka/premiumauthbypass).

## Requirements

- Java 25
- Paper / Folia server
- **AuthMe Reloaded** / **AuthMe ReReloaded**

## Commands

| Command               | Description                             |
| --------------------- | --------------------------------------- |
| `/amp accept`         | Link your current IP to your account    |
| `/amp revoke`         | Remove your current IP                  |
| `/amp revoke all`     | Remove all linked IPs                   |
| `/amp list`           | Show linked IPs (confirmation required) |
| `/amp about`          | Show plugin info                        |
